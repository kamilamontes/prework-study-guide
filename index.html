HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The decleration tells the browser what documents types it should expect when opened and know what to expect on the page
  • The opening tag represents the start of the HTML document. Inside of the tag is a attribute for language, which tells the browser the what laguage is diplayed on the the webpage
  • The heading element contains the metadata for the web applications
  • The title element defines the title for a webpage. This element is required and is extremely important for Search Engine Optimization (SEO)
  • The style element defines the CSS styles associated with a website
  • The base element defines the base URL for a webpage
  • The link element connects an external resource to the HTML document
  • The br element creates a line of empty space, or a line break, between two blocks of content
  • The meta element defines metadata such as the charset UTF-8 character set, description, keywords, author, and viewport(sets the defult view for the webpage)
  • The script element embeds client-side scripts in the HTML document
  • Metadata isnt seen by users, but it contains important information about accessibility, search engines and preformance
  • The elements of the HTML document include header, title, base and div
  • Elements that are nested inside other elements are called child elements. The element that holds the "children" elements are known as the parents. The "parent/child" language is often used by develpers to explain the relationship among elements in HTML
  • Semantic HTML elements describe the content and the role it plays on a webpage, (ex. body)
  • Non-semantic elements such as div create a division in the page, or span creates an inline division
  • Elements like images, brackets and links also made up the HTML structure
  • The `src` attribute defines the location, or source, of the image file
  • The `alt` attribute contains a text string that describes the appearance and functionality of the image

CSS

  • CSS is a programming language used by developers to define how webpage content is presented to users.CSS defines the colors, aesthetics, visual layout, and organization. We use CSS to define style rules for specific elements of an HTML page and format them using a wide range of style options
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the contentinside an element
  • inline CSS (Inline styles), apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only. They're useful for quick and permanent changes but are less flexible than external and internal style sheets, because each inline style must be edited individually whenever you make a design change
  • internal style sheet contains CSS rules for a webpage in the head section of the HTML file. The rules only apply to that page, and you can configure CSS classes and IDs, which are used to identify elements, to style multiple elements in the page code
  • The most common way to style a webpage is by using an external CSS style sheet. This is preferred in most cases because it allows developers to keep all their CSS rules in a separate file, which makes design changes easier
  • A selector (div) defines the element or attributes to which the rules, or declarations
  • Declarations contain two important components: the CSS property we want to apply and the value of the property. For example, the declaration color: blue; includes the CSS property color and the value for that property, blue
  • the h1 and h2 rule assignment is a positioning property, text-align one way to center a title or heading on a webpage
  • The img selector assigns property values to any image element in the markup. The display property is assigned with the block value, which assigns each image to start on a new line
  • The CSS rule for the ul elements declares the font, and the padding on the left of the unordered list
  • A link element creates the connection to the external file
  • The rel attribute specifies the relationship between the current document and the linked document resource
  • The href attribute specifies the location (URL) of the external resource. The href attribute is using a relative path
  • The period preceding card designates this as a class selector. Any HTML element assigned to the class, card, will gain these property assignments from the CSS

Git

  • Git is a free open source version control system that allows developers to collaberate on projects without overwriting each others work
  • It also allows developers to revert to pervious version of the code if a problem is discovered
  • Git status: checks what branch we are currently on
  • Git checkout -b branch-name: creates a new branch and switches to it
  • Git add -A: commands add modifications in the current working branch to the staging area. the -A flag indicates that we want to add all changes. The staging area informs git that the changes will be added in the next commit
  • Git pull origin main: identify a git command with the git keyword. the pull command to receives a branch's modifications into the local environment. The origin keyword indicates the source of the pull will be in the github repo. The main keyword indicates the branch
